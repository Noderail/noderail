What NodeRail Is

NodeRail is an intellectual lineage infrastructure platform. The NodeRail MCP Server is a read-only interface that allows AI assistants to query publicly available framework nodes, knowledge lineage records, and provenance metadata stored in the NodeRail repository.

What Data We Collect

The NodeRail MCP Server does not collect personal data.

  • No user accounts are required to use the MCP server
  • No personally identifiable information (PII) is requested, stored, or transmitted
  • No authentication credentials are collected
  • No usage logs tied to individual users are retained

The server reads from a locally cloned copy of the NodeRail public repository. All data it accesses is publicly available framework and knowledge metadata.

Data Transmission

All connections to the NodeRail MCP Server are encrypted via HTTPS/TLS. No data is stored beyond what is required to fulfill a single session request.

Third-Party Services

The NodeRail MCP Server does not connect to third-party APIs, advertising networks, or analytics services at runtime. It does not send user query data to any external party.

Your Rights

Because NodeRail does not collect personal data, there is no personal data to access, correct, or delete. If you have questions, contact us at info@noderail.org.

Changes to This Policy

We may update this policy as NodeRail's features evolve. Changes will be reflected by updating the effective date above. Continued use of the NodeRail MCP Server constitutes acceptance of the current policy.
