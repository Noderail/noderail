Node types

Four structured units of intellectual work

Field

Field Node

Defines a domain: scope, constructs, ethics, governance, and a constitution for contributions.

Framework

Framework Node

A reusable method, model, or operational playbook inside a field. Citable and forkable.

Measurement

Measurement Node

An instrument, index, or scale for measuring a construct. Versioned and linked to its field.

Project

Project Node

A capstone, case study, or pilot with a required Continuation Hook so work compounds.
